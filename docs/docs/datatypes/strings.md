# Strings

## GET

Fetches a value of a key.

```text title="syntax"
GET key
```

**Time complexity**: O(1)

## SET

Sets the value of a key

```text title="syntax"
SET key value
```

**Time complexity**: O(1)

## DEL

Deletes a key

```text title="syntax"
DEL key
```

**Time complexity**: O(1)
